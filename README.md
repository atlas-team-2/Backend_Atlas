# Проект: "Атлас народов"
## Описание проекта:
##### Проект представляет собой веб-платформу для изучения народов России. Пользователь может выбрать регион из списка и увидеть карту с народами, которые проживают в этом регионе. Пользователь может выбрать народ из списка и перейти к статье с информацией. Платформа позволяет пользователям создавать и редактировать материалы о народах, а также предлагать правки к существующим статьям. В статье отображается мини-карта, где показаны все регионы проживания выбранного народа.

## Инструкция по запуску
1. Клонировать репозиторий
bash
git clone https://github.com/atlas-team-2/backend_atlas.git
cd backend_atlas
2. Установить зависимости

Если используется venv:

python -m venv .venv

source .venv/bin/activate

.venv\Scripts\activate   # Windows PowerShell

pip install -e .
3) Запустить сервер
uvicorn main:app --reload --host 0.0.0.0 --port 8000

После запуска:

API: http://localhost:8000

Быстрая проверка

Откройте в браузере http://localhost:8000/docs и выполните запрос GET / 
— должен вернуться JSON со статусом сервиса.
