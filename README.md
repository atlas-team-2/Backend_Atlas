# Проект: "Атлас народов"

 Проект представляет собой веб-платформу для изучения народов России. Пользователь может выбрать регион из списка и увидеть карту с народами, которые проживают в этом регионе. Пользователь может выбрать народ из списка и перейти к статье с информацией. Платформа позволяет пользователям создавать и редактировать материалы о народах, а также предлагать правки к существующим статьям. В статье отображается мини-карта, где показаны все регионы проживания выбранного народа.
